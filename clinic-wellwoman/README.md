# Well Woman Clinic — Next.js Website

A modern, mobile-first clinic website built with Next.js 14, Tailwind CSS, and App Router.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Doctor Photo

Place the doctor's photo at:
```
/public/doctor.jpg
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Vercel auto-detects Next.js — click Deploy
4. Live in ~2 minutes

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Google Fonts (Playfair Display + Inter)
- No external UI libraries

## Sections

- Navbar (sticky, mobile responsive)
- Hero (WhatsApp + Call CTAs)
- About (clinic info + doctor photo)
- Timings & Location (table + Google Maps)
- Contact (form with success state)
- Footer (Nexlo Systems credit)
